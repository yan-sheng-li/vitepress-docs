# Nano文本编辑器

## 1. 简介
GNU Nano 是一款在 Linux 终端中运行的轻量级文本编辑器。它的设计目标是简单易用，相比 `vim` 或 `emacs`，学习曲线非常平缓，非常适合新手编辑配置文件或快速写点东西 。界面底部通常会显示常用的快捷键提示，对新手非常友好 。

## 2. 安装 Nano
大多数 Linux 发行版默认预装了 Nano。可以先在终端输入以下命令检查是否已安装：

```bash
nano --version
```

如果系统提示“未找到命令”，可以根据你的 Linux 发行版使用对应的包管理器进行安装：

| 操作系统 (Linux 发行版) | 安装命令                                      |
| :---------------------- | :-------------------------------------------- |
| **Ubuntu / Debian**     | `sudo apt update && sudo apt install nano -y` |
| **CentOS / RHEL**       | `sudo yum install nano -y`                    |
| **Fedora**              | `sudo dnf install nano`                       |
| **Arch Linux**          | `sudo pacman -S nano`                         |
| **openSUSE**            | `sudo zypper install nano`                    |

## 3. 基本操作流程

### 3.1 打开/新建文件
在终端中输入 `nano` 后跟文件名。如果文件不存在，Nano 会自动在当前位置创建一个新文件 。

```bash
nano my_first_note.txt
```

### 3.2 编辑文本
打开文件后，可以直接开始输入或修改文字。使用键盘的 **方向键（↑↓←→）** 来移动光标。默认情况下**不支持鼠标**操作 。

### 3.3 保存文件
在 Nano 中，**`^` 符号代表 `Ctrl` 键**。
- 按下 **`Ctrl + O`**（WriteOut）：这会触发保存操作。
- 按 **`回车（Enter）`** 确认文件名。界面底部会提示写入的行数 。

### 3.4 退出编辑器
- 按下 **`Ctrl + X`**（Exit）：试图退出 Nano。
- 如果文件在最后一次保存后有过修改，Nano 会询问是否保存：
    - 按 **`Y`**（是）：先保存再退出。
    - 按 **`N`**（否）：不保存直接退出。
    - 按 **`Ctrl + C`**：取消退出操作，返回编辑界面 。

> **流程总结**：编辑文件 -> `Ctrl + O` -> `回车` -> `Ctrl + X`。

## 4. 常用快捷键速查表

| 功能分类      | 快捷键         | 作用描述                                                     |
| :------------ | :------------- | :----------------------------------------------------------- |
| **通用**      | `Ctrl + G`     | 显示帮助文档（查看所有快捷键列表）                           |
|               | `Ctrl + X`     | 退出编辑器                                                   |
|               | `Ctrl + O`     | 保存文件 (Write Out)                                         |
| **编辑**      | `Ctrl + K`     | 剪切当前行（或选中的文本块）                                 |
|               | `Ctrl + U`     | 粘贴刚才剪切的内容                                           |
|               | `Alt + 6`      | 复制当前行（或选中的文本块）                                 |
|               | **`Ctrl + 6`** | **开始标记文本**：移动光标选中区域，选中后可用 `Ctrl + K` 剪切 |
| **查找**      | `Ctrl + W`     | 在文件中搜索关键词                                           |
|               | `Alt + W`      | 查找下一个匹配项（重复上一次搜索）                           |
|               | `Ctrl + \`     | 查找并替换文本                                               |
| **定位**      | `Ctrl + C`     | 显示当前光标所在的行列位置                                   |
|               | `Ctrl + _`     | 跳转到指定行号                                               |
| **撤销/重做** | `Alt + U`      | 撤销上一步操作                                               |
|               | `Alt + E`      | 重做撤销的操作                                               |

## 5. 进阶技巧与实用方法

### 5.1 显示行号
编辑配置文件或代码时，显示行号非常方便。
- **临时开启**：启动时加参数 `-l`。
    ```bash
    nano -l my_file.txt
    ```
- **永久开启**：编辑 Nano 的配置文件 `~/.nanorc`，添加一行 `set linenumbers`。修改后重启 Nano 生效 。

### 5.2 语法高亮
Nano 支持编程语言的语法高亮。可以使用 `-Y` 参数指定语言类型，例如编辑 Python 文件时启用高亮：

```bash
nano -Y python script.py
```
也可以将常用的配置写入 `~/.nanorc` 中，例如启用所有支持的高亮并设置 Tab 为 4 个空格 ：
```bash
include "/usr/share/nano/*.nanorc"
set tabsize 4
set tabstospaces
```

### 5.3 使用 `sudo` 编辑系统文件
当需要修改 `/etc/` 目录下的系统配置文件（如修改软件源、网络配置）时，必须拥有 root 权限。请使用 `sudo` 打开 Nano：

```bash
sudo nano /etc/apt/sources.list
```
如果不慎使用了普通权限打开并编辑了无法保存，可以使用 `Ctrl + X` 退出，然后重新运行上述带 `sudo` 的命令。
